# 奶龙美术馆 (Nailong Art Museum)

一个趣味艺术鉴赏互动网站，通过奶龙改编的世界名画，让用户在轻松有趣的氛围中猜测原画名称。

🔗 **在线体验**: https://lumia1998.github.io/nailong-museum/

## 这是什么？

奶龙美术馆是一个寓教于乐的艺术知识问答网站。网站展示了20幅"奶龙版"世界名画，用户需要根据画面猜测原作名称。

- 答对会播放奶龙跳舞庆祝视频
- 答错会播放奶龙哭泣视频
- 每道题配有"沉思奶龙"讲解原画背景知识

## 功能特性

- 🐉 20道世界名画选择题
- 🎬 答对/答错触发不同奶龙反应视频
- 💬 沉思奶龙气泡弹窗讲解
- 🌌 星月夜主题背景
- 🎵 舒缓背景音乐
- 📱 响应式设计（手机/电脑）
- ⛶ 全屏模式支持
- 🔐 用户登录/注册系统
- 📤 作品投稿功能

## 技术栈

- HTML5 + CSS3 + Vanilla JavaScript
- Supabase (后端服务)
- 响应式设计
- localStorage 进度保存

## 本地运行

```bash
git clone https://github.com/lumia1998/nailong-museum.git
cd nailong-museum
# 直接用浏览器打开 index.html 即可
```
