# nailong-museum 设计文档

## 项目概述
奶龙艺术鉴赏网站 - 通过奶龙改编的世界名画，让用户猜测原画名称，寓教于乐。

## 美学定位
**"古典美术馆 × 网络迷因"的碰撞美学**
- 深靛蓝 (#1a1a2e) + 奶龙黄 (#FFD93D) + 古董金 (#C9A227)
- Playfair Display（古典衬线）+ Noto Sans SC（现代无衬线）
- 深色纹理画布 + 油画笔触噪点背景

## 页面结构

### 1. 主页 (index.html)
- 沉思奶龙大图（呼吸动画）
- 标题："欢迎来到奶龙美术馆"
- 按钮："开始挑战"

### 2. 答题页 (quiz.html)
- 显示当前题目编号（第 X / 20 题）
- 奶龙改编名画图片
- 问题："这幅画改编自哪幅世界名画？"
- 4个单选选项
- "确认答案"按钮

### 3. 反馈页 (feedback.html)
- 答对：播放奶龙捧腹大笑视频
- 答错：播放忧郁奶龙视频
- 沉思奶龙 + 气泡对话框（100字讲解）
- "下一题"按钮

### 4. 通关页 (complete.html)
- 播放"你过关"视频
- 大字显示："可我觉得很神圣啊！"
- 可选：重新开始

## 技术栈
- HTML5 + CSS3 + Vanilla JavaScript
- 响应式设计（移动端优先）
- localStorage 保存答题进度
- 视频使用原生 `<video>` 标签

## 动画效果
1. 主页奶龙：呼吸动画（上下浮动 2s infinite）
2. 按钮hover：金色边框发光 + 轻微放大
3. 答题选项：选中时油画笔触填充效果
4. 气泡对话框：打字机效果逐字显示讲解
